SpreeFbtabViews
================

Adds support of mobile devices to Spree stores through jQuery Mobile.


Installation
============

1. Add following line to Gemfile: `gem 'spree_fbtab_views'`
1. Run `bundle install`
1. Run `rake spree_fbtab_views:install:assets`
